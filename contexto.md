# PROJETO: Landing Page High-End "Espaço Bem Me Quer"

## 1. Visão Geral
Landing Page para clínica de reabilitação com conceito "Bio-Digital Flow".
Objetivo: Conversão via WhatsApp e Agendamento Online.
Estética: Minimalista, luxuosa, fluida, animações orgânicas.

## 2. Tech Stack (Rigorosa)
- HTML5 Semântico.
- CSS3 Puro (Moderno: Grid, Flexbox, Variables).
- JavaScript (ES6+ Modules).
- Libs via CDN:
  1. GSAP + ScrollTrigger (Animações).
  2. Three.js (WebGL Hero).
  3. Lenis (Smooth Scroll).
  4. SplitType (Texto).

## 3. Design System
- Fonte Títulos: 'Playfair Display' (Serifa).
- Fonte Texto: 'Lato' ou 'Inter' (Sans).
- Cor Fundo: #F9F7F2 (Off-White/Creme).
- Cor Primária: #4A6C58 (Verde Sálvia Profundo).
- Cor Secundária: #D4A373 (Terracota Suave).
- Cor Texto: #2C3330.
- Bordas: 20px (arredondadas).

## 4. Diretrizes de Código
- Mobile First.
- Classes CSS seguindo padrão BEM (ex: .hero__title--large).
- Código limpo e comentado.
